# Verificador de Contratos — Portalia Desarrollos Inteligentes

Herramienta interna para verificar contratos con inteligencia artificial.

## Uso

Abre `index.html` en tu navegador o visita la página en GitHub Pages.

## Tecnología

- HTML, CSS, JavaScript (archivo único)
- PDF.js para lectura de contratos en PDF
- API de Anthropic (Claude) para análisis con IA
